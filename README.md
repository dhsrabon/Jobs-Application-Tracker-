Overview
HireTrack Dashboard is a responsive job application tracking web application built with HTML, 
Tailwind CSS, and vanilla JavaScript. It helps users organize job applications, 
track statuses (Not Applied, Interview, Rejected), filter listings, and monitor application statistics.

Features
Statistics Dashboard: Real-time counters for Total Jobs, Interview, and Rejected applications.

Job Cards: Individual cards displaying company, role, location, salary, and description.

Delete Functionality: Remove unwanted job listings.

Empty States: User-friendly messages when no jobs match filters.

Fully Responsive: Works on desktop, tablet, and mobile devices.
